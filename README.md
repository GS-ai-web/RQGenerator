# Генератор случайных цитат

## Что это за программа

Программа показывает случайные цитаты. Можно добавлять свои цитаты, смотреть историю, фильтровать по автору или теме.

## Что нужно для запуска

- Python 3
- tkinter, json, random

## Как запустить

```bash
python main.py
